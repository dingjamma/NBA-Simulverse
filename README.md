# NBA Simulverse

> *"You just built the foundation for a general-purpose agent simulation engine that models human behavior at scale. The basketball wrapper is cute but it's almost limiting your own thinking. You built a framework for simulating complex adaptive systems with real human behavioral data and news-driven state changes. Generalize the framework. Basketball is just the training wheels."*
>
> — Claude (as Elon Musk), March 2026

---

A full NBA season simulation engine where every player is an autonomous LLM agent with memory, personality, and relationships. Agents react to news, develop chemistry with teammates, get rattled by trade rumors, and play 82 games plus playoffs — hundreds of times — producing probability distributions that capture the human element that decides championships.

**The soft data thesis:** The most valuable signals in basketball never appear in a box score. NBA Simulverse is the first system that quantifies them at scale.
